# Installation

## Prerequisites

- **Bun 1.3.0 or higher** — Check your version: `bun --version`. Install via [bun.sh](https://bun.sh).

## Install

```bash
bun add -g spok
```

## Verify Installation

```bash
spok --version
```

## Next Steps

After installing, initialize Spok in your project:

```bash
cd your-project
spok init
```

See [Getting Started](getting-started.md) for a full walkthrough.
